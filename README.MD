# Personal Website

This repository contains the source code for my personal website, which showcases my portfolio and blog.

The site is built with [Hugo](https://gohugo.io/), a fast and modern static site generator, and styled using the [Congo](https://github.com/jpanther/congo) theme.

Deployment is automated through **GitHub Actions** and published via **GitHub Pages**.

## Local Development

To run the website locally, you need [Hugo Extended](https://gohugo.io/installation/) installed.

```bash
# Start the local server
hugo server
```

Once running, open [http://localhost:1313](http://localhost:1313) in your browser to preview the site.
